## 👨‍💻 About Me

I'm **Tobias Viana**, a Software Engineering student at **Inteli** building intelligent and scalable digital solutions that solve real-world problems.

My journey spans the full product lifecycle — from problem scoping and architecture design to implementation, testing, and cloud deployment — always aiming for **efficiency, scalability, and measurable impact**.

**Core areas of focus:**

- 🤖 **AI & Machine Learning** — predictive models, recommender systems, AI agents
- 📊 **Data Science** — exploratory analysis, feature engineering, actionable insights
- ☁️ **Cloud & DevOps** — AWS, Docker, deploy pipelines, cloud-native architecture
- ⚙️ **Automation** — RPA, API integrations, workflow optimization
- 🌐 **Full-Stack Development** — web platforms, REST APIs, databases, auth systems

> Projects in partnership with companies across healthcare, gaming, and enterprise sectors — applying tech to challenges that matter.
